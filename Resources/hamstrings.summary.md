# Hamstring Training Tips (Hypertrophy) Summary

## Best Exercises to Model
- Stiff-legged/RDL-style deadlift
- Good morning
- 45-degree back extension (ham/glute bias)
- Lying leg curl
- Seated leg curl
- Glute-ham raise

## Programming Notes
- Core movement buckets: hip hinge and knee flexion curl.
- Lengthened-position loading is a major emphasis.
- Balanced programming should include both hinge and curl categories.
