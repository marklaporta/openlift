# Back Training Tips (Hypertrophy) Summary

## Best Exercises to Model
- Chest-supported row / T-bar row / bent-over row
- Pull-up variations
- Lat pulldown variations (grip-specific)
- Straight-arm pulldown
- Machine or dumbbell pullover

## Programming Notes
- Core movement buckets: horizontal pull, vertical pull, shoulder-extension/pullover.
- Include grip variants as separate exercise options where useful.
- Weekly programming should distribute both row and vertical pull patterns.
