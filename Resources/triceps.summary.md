# Triceps Training Tips (Hypertrophy) Summary

## Best Exercises to Model
- Cable pushdown
- Overhead cable extension
- Skullcrusher / lying triceps extension
- Dip / assisted dip
- Close-grip pressing variation

## Programming Notes
- Core movement buckets: overhead extension, pushdown, compound press/dip extension.
- Long-head emphasis is often tied to overhead patterns.
- Include elbow-friendliness metadata for exercise substitutions.
