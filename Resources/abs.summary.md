# Ab Training Tips Summary

## Best Exercises to Model
- Machine crunch
- Cable rope crunch
- Hanging knee raise / leg raise
- Slant-board sit-up
- V-up

## Programming Notes
- Core movement buckets: spinal-flexion crunch and hanging raise patterns.
- Technique emphasis is controlled trunk motion and meaningful ROM.
- Include load-progressible ab options (machine/cable) in defaults.
